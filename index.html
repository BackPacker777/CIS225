Description

CIS 225 Introduces students to the Java programming language. Object oriented concepts & methodologies are taught and practiced. The course focuses directly on UML, classes, objects, fields, constructors, methods, enums, Java control structures + logic, operators, coding style, Java library classes, collections, GUI programming, user and file I/O, and program documentation including Javadoc. This course prepares the student for the Oracle Java SE 8 Programmer I exam. Additionally, an introduction to Android development is explored at the end of the course.
